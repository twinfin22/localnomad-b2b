| **FIMS 필드명** (예: "학번", "비자만료") | ❌ | AI 매핑 엔진 훈련 데이터 | 양식 또는 법무부 FIMS 매뉴얼 | W0 D10 |
| **FIMS 데이터 타입 & 길이** (예: VARCHAR(50)) | ❌ | 입력 검증 규칙 | 양식 또는 대학 담당자 문의 | W1 |
| **FIMS 제출 프로세스** (온라인? 오프라인?) | ❌ | 변동신고 워크플로우 설계 | 법무부 홈페이지 또는 담당자 인터뷰 | W0 말 |
| **FIMS 제출 기한** (15일?) | ⚠️ 확인됨 (15일) | 알림 로직 설정 | 출입국관리법 또는 담당자 | W0 |
| **FIMS 거부 사유 Top 3** | ❌ | 검증 규칙 우선순위 | 대학 담당자 또는 법무부 통계 | W1 |
| **FIMS 정기 보고 양식** | ❌ | 자동 보고서 생성 기능 설계 | 파일럿 대학 요청 | W0 D10 |

```
제목: FIMS 입력 양식 및 담당자 자문 요청

내용:
저희는 대학의 FIMS 입력 업무를 간소화하는 B2B SaaS를 개발 중입니다.

다음 자료를 요청드립니다:

추가로 담당자님과 30분 전화 통화를 원합니다:
- 현재 유학생 관리 프로세스
- FIMS 이외의 시스템 (학사시스템, 보고 시스템 등) 연동 여부
- 가장 원하는 기능
```
