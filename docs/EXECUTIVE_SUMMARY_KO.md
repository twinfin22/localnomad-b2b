## Local Nomad B2B 비자 대시보드

**작성**: 2026-02-13 | **대상**: CEO, 경영진

## 핵심 발견사항

### Phase 1 실행 가능성: **⚠️ 조건부 진행 (위험 높음)**

현재 **시장 조사 보고서와 Phase 1 스펙 문서는 전략적 방향성은 명확하나, Phase 1 실행에 필수적인 실제 데이터(경쟁사 가격, 대학 선정 기준, 법적 검증, 사용자 인터뷰)가 거의 없는 상태**입니다.

**즉시 조치 없이 Phase 1을 진행할 경우:**
- 파일럿 대학 확보 실패 (타겟 리스트 없음)
- 가격 협상 불가능 (경쟁사 가격 미파악)
- 법적 리스크 노출 (변호사 검토 전)
- 기술 검증 불완전 (프로토타입 테스트 전)

**예상 영향**: 6주 일정 중 2-3주 지연, 또는 전략 재검토 필요

## 최악의 시나리오 & 대응안

### Scenario A: MOU 0건 (W0 실패)

**신호**: D6 종료 시 파일럿 대학 미확보
- **즉시 조치**: CEO 직접 방문영업 (서울권 3곳)
- **대안 1**: KAFSA 회장/명예회장에게 추천 부탁 (인맥 활용)
- **대안 2**: 교육부 국제교육진흥원 담당자 → 대학 추천
- **리스크**: 1주 지연 → W1 개발 시작 지연

### Scenario B: FIMS 양식 미확보 (W0 실패)

**신호**: 파일럿 대학이 "FIMS 양식을 주기 어렵다" 거절
- **대응**: "그러면 먼저 프로토타입을 보여드리고, 양식은 나중에 확보하겠습니다"
- **개발 진행**: 범용 엑셀 내보내기부터 시작, FIMS 양식은 확보 즉시 템플릿 추가
- **리스크**: F1-07 (FIMS 내보내기) 개발이 W3-W4로 미루어짐

### Scenario C: 경쟁사 가격 0건 (W0 실패)

**신호**: HireVisa 고객사 5곳 모두 "계약상 비밀" 거절
- **대응**:
  - 간접 지표 사용: "대학 IT 예산 = 평균 5000만원/년" (업계 일반)
  - 벤치마크: 유사 B2B SaaS (HRIS 100-500명 기업용) 가격 = 월 50-300만원
  - 대학 규모 조정: 학생 400명 = 중소기업 규모 → 월 80-120만원 추정
- **파일럿 가격**: "첫 3개월 무료, 이후 월 50만원" (시장진입 가격)
- **리스크**: 정가 책정이 임의적 → Phase 2 이후 가격 조정 필요

### Scenario D: 법적 검증 지연 (W1-W2)

**신호**: 변호사가 "추가 검토 필요, 3주 소요" 답변
- **대응**:
  - 개발은 계속 진행 (기술적 로직은 일단 중립적)
  - "개인정보 처리" 부분만 보류 (위탁 계약서 완성까지)
  - 파일럿 서명 전에 법적 결론 필수
- **리스크**: 파일럿 온보딩(W5) 지연 가능

## 핵심 메시지 (의사결정)

### 권장: **"W0를 2주 연장하고, 필수 데이터 4가지 확보 후 개발 시작"**

**이유**:
1. **데이터 없이 개발 → 방향 오류 리스크 높음**
   - 가격 미정 → 파일럿 협상 불가
   - 대학 미선정 → 개발 기준 없음 (FIMS 양식 미정)
   - 법적 미검증 → 수개월 후 기소/행정처분 가능

2. **W0 2주 투자 >> 나중의 피벗 비용**
   - 인터뷰, 자료수집: 100~300만원
   - 법무 자문: 200~500만원
   - 총 500만원 vs. 개발 재작업 비용(수천만원)

3. **현재 스케줄은 "happy path" 기반**
   - 모든 가정이 정확하다고 가정
   - 실제로는 50% 이상 실패율 예상

### 대안: **"W0는 현재대로 진행하되, W1-W2에 병렬로 조사 계속"**

**조건**:
- [ ] W0 D10: MOU 1곳 + FIMS 양식 최소 1종 + 경쟁사 가격 2곳 이상 → Gate 통과
- [ ] Gate 통과 못 하면 → W1 일시중단, 전략 재검토 (1주)
- [ ] W1은 개발 + 병렬 인터뷰/조사 진행 (BizDev는 계속 영업)

## 최종 결론

**현재 상태**: 전략적 방향은 탁월하나, 실행 데이터 부족

**즉시 필요한 것**:
1. ✅ 파일럿 대학 1곳 MOU (없으면 모든 게 멈춤)
2. ✅ HireVisa 가격 데이터 (사업성 검증)
3. ✅ FIMS 양식 (개발 기준)
4. ✅ 법적 검증 (리스크 대비)
5. ✅ 사용자 인터뷰 (방향 재확인)

**위험 수준**: 🔴 **높음** (현재 상태로는 실패 확률 50%)
**완화 전략**: 🟢 **가능** (2-3주 집중 조사로 70% → 80% 성공률 달성)

**권장 결정**: **W0 연장 + 필수 조사 완료 후 개발 시작** ⏱️
